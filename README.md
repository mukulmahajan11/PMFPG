# Predictive Maintenance for Power Grids — Complete 6-Task Project

This project implements an end-to-end predictive maintenance workflow for utility assets using synthetic power-grid sensor data and outage logs.

## Code Files
- `src/generate_synthetic_data.py`
- `src/task1_data_ingestion_cleaning.py`
- `src/task2_sensor_preprocessing_feature_engineering.py`
- `src/task3_outage_log_analysis_label_generation.py`
- `src/task4_ml_model_selection_initial_training.py`
- `src/task5_model_optimization_validation.py`
- `src/task6_market_mood_index_definition.py`
- `src/run_all_tasks.py`

## Run
```bash
pip install -r requirements.txt
python src/run_all_tasks.py
```
